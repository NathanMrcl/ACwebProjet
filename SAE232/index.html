Objectif du site

Cette SAE a pour but de nous faire exploiter les données provenant de capteurs ( dans notre cas les capteurs de température) répartis dans les bâtiments de l’IUT. Il faut réussir à proposer une interface dédiée à ces données. C’est pourquoi nous avons créé ce site Web.

Contrainte Technique

Pour réaliser ce projet nous devons respecter des contraintes. Une grosse partie de ce projet est réalisée à partir d’une VM Ubuntu, de plus il faut utilisé des chaînes de traitement via des conteneurs Docker comme Mosquitto, Node-RED, InfluxDB et Grafana. Pour ce qui est du site Web dynamique nous devons utiliser une base de données MySQL, faire de la gestion de compte, planifier des tâches avec du code Bash et PHP et bien sûr présenter les données et métriques récupéré de deux bâtiments et de deux salles par bâtiments.
